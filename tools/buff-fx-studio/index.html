使用说明 / How to use
  1. ① 选素材：emoji 库里挑一个，或上传自定义 PNG/SVG（推荐透明背景方形）
  2. ② 选模板：9 种动画模式，移上去看简介
  3. ③ 调参数：右边实时预览同步
  4. ④ 看预览：网格背景验证透明度，下方深色块模拟 token 位置
  5. ⑤ 点生成：首次会下载 ffmpeg.wasm（~30 MB，缓存一次后不再下）

生成的 .webm 可以放进 OBR Suite 插件 public/buff-fx/，或者作为 OBR Image item 的 url 字段使用。
