Buff FX preview · alpha sanity check

Each effect is shown twice — on a CHECKERBOARD bg (alpha shows the pattern) and on a flat BLUE bg. The "token" underneath is just a CSS gradient so you can see the effect sit on top of it like it would on a real OBR token. If alpha works correctly, you'll see the BACKGROUND THROUGH the WebM frame, with only the emoji sprites visible. If alpha is stripped, the whole 192×192 frame will appear as an opaque black square covering the token.
